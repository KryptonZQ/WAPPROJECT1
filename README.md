# 🎮 Retro Game Vault

## 📌 Project Description

Retro Game Vault is a web application that allows users to search and explore video games from a large database. Users can filter and sort results, view game details, and save their favorite games to a wishlist.

The goal of this project is to demonstrate the use of JavaScript, API integration, and modern UI design.

---

## 🌐 API Used

This project uses the **RAWG Video Games Database API**.

API Link: https://rawg.io/apidocs

---

## ✨ Features (Planned)

* 🔍 Search games by name
* ⏱️ Debounced search input for optimized API calls
* 🎮 Filter games by platform (PC, PlayStation, Xbox, etc.)
* 🔼 Sort games by rating or release date
* ⭐ Add/remove games from Favorites (stored in Local Storage)
* 📊 Compare two games side-by-side (advanced feature)
* 🎨 Responsive grid-based UI with game cards

---

## 🛠️ Technologies Used

* HTML
* CSS (or Tailwind CSS)
* JavaScript (Vanilla JS)
* Fetch API
* Local Storage

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/retro-game-vault.git
   ```

2. Navigate into the project folder:

   ```bash
   cd retro-game-vault
   ```

3. Open `index.html` in your browser

4. (Optional) Add your RAWG API key in the JavaScript file
